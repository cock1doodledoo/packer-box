packer-box
==========
build my base-box using the [packer]

### specs of builded box
* [ubuntu] 14.04
* Japanese keyboad

* [virtualbox] 4.3.20
* [vagrant] 1.7.1
* [ruby] 2.1.3
* [rake] 10.1.0
